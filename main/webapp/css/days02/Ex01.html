[글꼴(font)]
	- 웹 사이트에 적합한 글꼴을 선택하는 것은 중요.
	- 읽기 쉬운 글꼴, 크기, 색상 중요
	- 웹 안전 글꼴, 모든 브라우저에서 보편적으로 설치
	- 크기
		상대크기 : 브라우저 기본 글꼴 크기 - 16px
		1em : 기본 브라우저의 글 크기와 동일하다. ( 16px )
				 body 	 p 	   div
		16px    0.5em  0.5em  0.5em
				8px    4px    2px
				
					 
		16px    0.5rem 0.5rem 0.5rem
				8px    8px    8px
		절대크기 : cm, mm 등등
