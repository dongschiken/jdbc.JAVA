[ display 속성 ]
	1. 레이아웃(배치)을 제어하기 위한 가장 중요한 css 속성
	2. 모든 요소의 기본값 : block, inline
	  block: div, p, header, hr, hn 등등 ( 특징 : 항상 새로운 라인에서 시작 //너비가 부모 너비의 100%로 설정된다.  )
	  inline: span, a, img 등등 ( 특징 : 필요한 내용만큼만 너비가 설정된다. )
	3. display => 요소가 표시되는 여부를 지정하는 속성
	4. display : none; ==> 숨기기(요소 삭제는 아니고 안보이게 처리하는 것)
