[ overflow 속성 ]
	- 요소의 내용이 너무 커서 지정된 영역에 맞지 않을 경우
	  그 내용을 자를지, 스크롤바를 추가 할지 등등을 지정하는 속성
	- visible(기본값) : 내용이 넘치더라도 자르지 말고 보여준다.
	- hidden : 잘라낸다.
	- scroll : 내용이 넘치는 유무에 상관없이 스크롤바를 추가
	- auto   : 내용이 넘칠 경우에만 스크롤바가 추가.
	(주의사항) overflow 속성은 높이가 지정된 블럭요소에만 적용된다.
