[ css 선택자를 속성(attribute) 사용 ]
	1. 속성명만 사용 => input[type] (이 태그는 type속성을 가지고 있지 않아서 스타일에서 제외된다.)
	2. 속성명 = 속성값
	3. 속성명 ^= 속성값 => WHERE title LIKE 'flower%';  // 플라워로 시작하는 속성
	4. 속성명 $= 속성값 => WHERE title LIKE '%flower';  // 플라워로 끝나는 속성
	5. 속성명 *= 속성값	=> WHERE title LIKE '%flower%'; // 플라워가 있는 모든 속성
	6. 속성명 ~= 속성값 =>  						    // 한 개의 단어가 플라워인 속성
	7. 속성명 |= 속성값 =>
