[css 특이성]
	- 동일한 요소를 가리키는 두개 이상의 css 규칙이 있는 경우
	가장 높은 특이성을 가진 선택자에 의해 css 적용된다.
	- 특이성 ? 스타일 적용 점수/우선 순위
	- (암기)
		└ 태그명 1점
		└ 클래스명 10점
		└ 아이디명 100점
		└ 태그명 + 아이디명 101점
		└ 태그명 + 아이디명 + 클래스명 111점
		
		인라인 css 적용 방식 : 1000점
		

		



	css 선택자(selector)
	1. 태그명( p, a등등)
	2. #ID명
	3. .클래스명
