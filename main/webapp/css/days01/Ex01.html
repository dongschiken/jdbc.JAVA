[CSS]
	- Cascading Style Sheets 약어
	  계단식
	  떨어뜨리다
	  작은폭포
	  단계적으로 행하다.
	- ㄱ. 기초적인 화면 구성 => html
	- ㄴ. 문서 스타일을 먹이는 언어 => css
	- 스타일 일괄적으로 관리 ( 유지 보수 용이 )
	- css는 html 요소가 화면, 종이, 기타 미디어에 표시되는 방법을 지정
	- 여러 웹 페이지의 레이아웃을 한 번에 제어할 수 있다.
	- 외부 스타일 시트 ( .css )사용
		└ 여러 웹 페이지 전체에 스타일 일괄적으로 적용할 수 있다.
	- css 형식( 구문 ) // 스타일 적용할 대상 == 선택자 == selector
		└ 스타일 적용할 대상{
			속성명: 속성값
			속성명: 속성값
			속성명: 속성값		
			html에서 단위를 생략하면 기본 px
			css에서 단위를 생략하면 적용 x
		}
	- 
	

	css 선택자(selector)
	1. 태그명( p, a등등)
	2. #ID명
	3. .클래스명
