[ css 변수 사용 - var() 함수 ]
	var() 함수 : css 변수의 값을 사용하는 함수(값을 할당하는 함수)
	- CSS변수는 DOM에 액세스 할 수 있다.
	- 로컬 또는 전역 범위의 변수 생성
	  -> js 사용 변수 변경
	  -> 미디어 쿼리를 기반으로 변수 변경할 수 있다.
	- css 변수 : 디자인 색상에 관해서 사용하면 좋다.
	예) 텍스트 박스
	   여러곳에 동일한 배경색을 설정하고 싶을 때. 
	   텍스트 박스 input[type=text]{ bc=val(css변수)}
	   상자 div, p {bc=val(css변수)}
