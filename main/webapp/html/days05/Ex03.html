[html 코딩 큐칙]
	1. <!DOCTYPE html> : 선언문, DOCTYPE 대문자 권장
	2. 태그(요소) : 소문자 권장
	3. 속성명="속성값"
	   속성명='속성값'
	   속성명=속성명
	4. 모든 html 요소의 닫기태그는 꼭 필요하진 않다. (그래도 닫기 태그를 하는 것이 좋다.)
	5. style="width: ??; height: ??;"
	6. 주의사항 : 요청하는 url은 대소문자 구분한다.
	http://localhost/webPro/html/ex01.html
	7. <a href="www.naver.com">naver</a>
		속성명 = 속성값 X
		속성명 속성값 O
	8. 탭키를 사용하지 말고 스페이스 여러번 사용. (주의사항 : 이유없는 공백, 빈줄, 들여쓰기 X)
	9. <head>필수 X</head>
	   <body>필수 O</body>
	10. 언어선언 O, 검색엔진과 브라우저에 알려주기 위해
	<html lang="en-us"></html>
	11. 뷰포트? : 웹페이지에서 사용자가 볼 수 있는 영역. 
		다양한 장치마다 뷰포트가 다르다.
		뷰포트 제어 -> meta태그 추가
		width=device-witdh 페이지의 크기는 장치 화면 너비
		initial-scale=1 : 브라우저가 페이지를 처음 로드할 때 초기 확대/축소 수준 설정.
	12. 외부 css 파일 연결할 때 
	<link href="sample.css" rel="stylesheet">
	14. 외부 js 파일을 연결할때
	<script src="sample.js"></script>
	15. .css / .js / .html or .htm
